Fundamentos:

1 - Identação em Python é considerada para bloco de códigos

```python
if 5 > 2:
    print ("5 é maior que 2")
```
Sem a identação no código acima, Python vai apresentar um erro por não conseguir interpretar o bloco de códigos.
